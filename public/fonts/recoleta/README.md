# Recoleta Font Files

Place the following Recoleta font files in this directory:

- Recoleta-Regular.woff2
- Recoleta-Medium.woff2
- Recoleta-SemiBold.woff2
- Recoleta-Bold.woff2
- Recoleta-Black.woff2

Once placed, uncomment the @font-face block in `src/app/globals.css`
or the localFont config in `src/app/layout.tsx`.
